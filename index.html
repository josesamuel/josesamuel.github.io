Simple open source tools

Remoter

An alternative to Android AIDL for Android Remote IPC services using plain java interfaces

RxRemote

Rx Observable across android remote process.

CarbonCopy

Create Carbon Copies (POJO data classes) from existing classes using annotation

Log Viewer

Android Studio Plugin with better log viewing capability.

ServiceConnector

Bind Android services and callbacks to fields and methods.
